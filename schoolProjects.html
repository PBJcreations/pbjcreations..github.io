Programming With OpenCV Face Detection

Image Processing

During my Image Processing Class I took at UNO, I learned a lot about how images and videos are compressed through the use of kernels. These kernel algorithms can create blur effects, hue shifts, saturation adjustments, auto-contrast, etc. by tweaking a small group of pixels at a time. I was eager to experiment with AI and Face detection using a popular API for Python called OpenCV. Essentially this contains resources for AI facial pattern detection. My class project took a PNG photo of a mustache and aligned it over a live video feed of someone's face by blending the alpha layers. This was a cool little project to explore a rising sector of technology in digital image processing and Artificially Intelligent programs.

Intro to Game Programming

Tanks Game

For my final project of the course, I created a 2-player versus tank game. I incorporated collision detection algorithms for points and rectangles like we learned from class. There are 5 random game levels. The first person to get 5 points wins. The way this html game works is by calculating all the vertices (points) of each shape, then draws them in with certain colors. It then receives user input from the keyboard to update the locations of each vertex. It repeats this process faster than the human eye can detect. For collisions, the game calculates if a point is intersecting another object and handles it according to the conditions at hand.

Tron Game

Earlier in the semester, I really wanted to understand how html and JavaScript worked to run a game. I followed a brief tutorial on the subject, resulting in a clone of the classic game, "Snake". After running wild with curiosity playing with the code, I discovered how to have two players at once. I also discovered how to create point-to-point collision detection for both players. After I got the base game down, I spent a great deal exploring the stylization and formatting of the game aesthetics. I also discovered I could create buttons on the side of the screen so that you could play it on your phone. However, due to the limitations of my knowledge of HTML, only one button could be pressed at one given moment of time.
